# ProQuest TDM Studio: Information, Tutorials, and Workshops

[ProQuest TDM Studio](https://about.proquest.com/en/products-services/TDM-Studio/) is a web platform for running text analyses on thousands of ProQuest materials, including but not limited to, ProQuest Dissertations & Theses and the New York Times.

Getting started
---------------

* Please [visit our guide](https://mdl.library.utoronto.ca/technology/text-data-mining-software/ProQuestTDMStudio) for details on how to access ProQuest TDM Studio Visualizations and Workbench
	+ [Create an account](https://tdmstudio.proquest.com/createaccount) for TDM Studio Visualizations using your University of Toronto email address
	+ [Apply](https://bit.ly/3olVWwT) for Workbench access using your Visualizations account
* Please consult the [TDM Studio Official User Guide](https://proquest.libguides.com/tdmstudio/home) for additional information
